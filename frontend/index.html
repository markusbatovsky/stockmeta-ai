Konfigurácia pripojenia k backendu a preferencie.

🔌 Pripojenie k backendu

Backend beží na http://localhost:8000 po spustení Docker Compose. API dokumentácia na http://localhost:8000/docs

⚡ Rýchly štart backendu

# Prejdite do priečinka projektu
cd stock-metadata-tool
# Skopírujte a nastavte env
cp .env.example .env
# Spustite Docker Compose
docker-compose up -d
# API beží na http://localhost:8000
